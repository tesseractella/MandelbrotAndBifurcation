Mandelbrot Set visualizer

The Mandelbrot Set is a fractal produced by iterating the function fc(zi+1) = >zi2 + c , over the complex plane, with the constant c being the point on the plane and z0=0 .

The Mandelbrot Set seen below iterates over the complex plane for z=a+bi for all a∈[-2,2] and b∈[-2,2] . Here, all points in the set are seen in black, meaning that regardless of the number of iterations performed, the function never diverges to infinity. Otherwise the colors represent the number of iterations for a point to escape the set.

The Mandelbrot set is actually an extension of the bifurcation map, produced by the logistic equation xn+1=f( xn)=rxn (1-xn) with the parameter r. This models how the behavior of a dynamic system changes as an result of the parameter. This is commonly used in modeling population dynamics and in chaos theory. It is also used in computer systems as a pseudorandom number generator. As r increases, there is successive period doubling, wherein the value of the function oscillates between 2 values, then 4 values, then 8 values and so on.

For r<3, there is one attractor point, which bifurcates at r=3. Bifurcations occur until until r=3.57 after which, the system divulges into chaos (though it is not always chaotic after this point).

Bifurcations reflect chaotic systems, wherein the behavior is highly sensitive to inital conditions and a small parameter change causes a qualitative change in the system. Here, as r increases, the value of the system, once at equilibrium, will diverge. For a small enough r, there is always an attractor, and thus the system is stable around one value. At r=3.57, there is periodic doubling, splitting the equlibrium into a stable cycle of 2, and as r increases, doubling occurs again and again at an accelerating rate, eventually leading to a non-cyclic, chaotic system.
